Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This project identifies the top 10 bacterial species in each volunteer's belly button. The dropdown button allows the user to choose which volunteer is viewed. The "Demographic Info" section provides personal information about the volunteer, whereas the bar chart shows the top 10 bacterial species and the quantity of those bacterial species for the volunteer. The gauge chart shows how frequently the volunteer scrubs the belly button per week, whereas the bubble chart shows the bacteria culture per sample.
